This folder contains the Matlab and C++ code for the colour transfer algorithm detailed in 'Robust Registration of Gaussian Mixtures for Colour Transfer'.


You can download this source code for free, use and change it as
you like, but please cite our paper:

Robust Registration of Gaussian Mixtures for Colour Transfer
Mairead Grogan and Rozenn Dahyot
Arxiv 2017
